# Research Agent

Simple research agent package with:

- CLI usage from CMD
- optional desktop launcher
- Obsidian vault reporting

## Install

```powershell
pip install -e .
```

## Run from CMD

```powershell
research-agent --query "latest AI news"
```

## Run as module

```powershell
python -m agent --query "latest AI news"
```

## Run desktop app

```powershell
research-agent --app
```
